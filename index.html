FxGrid

Minimalist flexbox grid system

Basic Usage

Just add css class class="col device-width"

<div class="row">
  <!-- <div class="col device-width"> -->
  <div class="col desktop-12">
    <div class="box"></div>
  </div>
</div>

Responsive

You can modify the size on the column on different devices.

Syntax:
<div class="row">
  <div class="col desktop-4 tablet-6 mobile-12">
    <div class="box"></div>
  </div>
</div>

Auto Width

Auto sizing the column.

Syntax:

Use auto to make column width auto

<div class="row">
  <div class="col auto">
    <div class="box"></div>
  </div>
</div>
